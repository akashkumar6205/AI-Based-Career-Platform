# 🛡️ CareerShield AI

**A Smarter, Safer Career Guardian for Students**

CareerShield AI is an all-in-one career companion designed to protect students and recent graduates from the hidden dangers of the modern job market. It detects fake job postings, flags phishing emails, optimizes resumes for ATS (Applicant Tracking System) bots, and coaches users through mock interviews—all powered by AI.

## ✨ Features

### Pillar 1: The Shield (Security)
- **Fake Job & Internship Detector**: Cross-references company registration databases, URL history, and language patterns to expose data-harvesting scams.
- **Spam & Fake Email Scanner**: Dissects email headers, scans for malicious links, and verifies sender domains to protect against phishing attempts.

### Pillar 2: The Sword (Growth)
- **AI Resume & ATS Optimizer**: Analyzes your resume against target job descriptions, providing precise ATS match scores, keyword suggestions, and formatting tweaks.
- **AI Career Coach & Mock Interviews**: Maps personalized career paths and provides realistic audio/text mock interviews with real-time feedback.

### For Universities & Placement Cells
A dedicated University Dashboard that empowers placement officers to screen recruiters, auto-scan placement emails for fraud, and track student readiness using AI analytics.

## 🚀 Tech Stack

- **Frontend Framework**: React (Bootstrapped with Vite)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion (Scroll animations, spring physics, and micro-interactions)
- **Routing**: React Router DOM

## 🛠️ Getting Started

Follow these steps to run the application locally.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository** (if applicable) or navigate to the project directory:
   ```bash
   cd "New folder (2)"
   ```

2. **Navigate to the frontend folder**:
   ```bash
   cd frontend
   ```

3. **Install the dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

5. **Open your browser**: 
   Navigate to `http://localhost:5173/` to view the application with all interactive animations.

## 🎨 Design & Aesthetics
The UI/UX focuses on a modern, premium dark-mode aesthetic utilizing "glassmorphism", neon-emerald and blue gradients, and buttery-smooth micro-interactions driven by `framer-motion`. The design emphasizes trust, security, and cutting-edge technology.
